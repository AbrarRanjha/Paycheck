# Paycheck
A production-ready Express.js REST API built with a modular domain-based structure, supporting payments, sales data, refunds, payouts, commission splits, reporting, dashboards, email history, and error logging. Includes Docker, Nginx, ESLint, and Prettier setup.



## Project Overview

This project is a modular and scalable backend API built with Node.js and Express.
It follows a domain-based folder structure where each business feature is isolated
into its own module for better maintainability and scalability.

### Key Features
- User management
- File uploads
- Sales data management
- Refund and early payment handling
- Payout processing
- Client commission splits
- Employee reports
- Dashboard APIs
- Email history tracking
- Centralized error logging

### Tech Stack
- Node.js
- Express.js
- Docker & Docker Compose
- Nginx
- ESLint & Prettier
- Environment-based configuration

### Architecture Highlights
- Modular route-based structure
- Clean separation of concerns
- Production-ready configuration
- Easy to scale and extend
